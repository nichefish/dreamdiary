## 개인 프로젝트: dreamdiary
###
#### 개요:
일기, 꿈, 기타 개인기록 정리. 글분류 및 트랙백, 태그로 연관된 정보를 동적으로 연관지어 관리.


#### 사용 스택: 

* Java 11
  * Spring Boot (2.7.18) 
  * Spring Security
  * Lombok
  * Mapstruct
* Gradle
* MariaDB
* Spring Data JPA / MyBatis
* Apache Freemarker 
* Javascript, jQuery, Handlebars.js
* Bootstrap 5 (Metronic 8)
* CSS / SCSS

### "All classes in the hierarchy must be annotated with @SuperBuilder."
